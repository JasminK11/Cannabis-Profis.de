# 🚀 SCHNELL-DEPLOY ANLEITUNG

## ⚡ EINFACHSTE METHODE (3 Klicks)

### 1. Gehe zu Vercel
👉 **https://vercel.com/new**

### 2. Importiere Repository
- Klicke auf **"Import Git Repository"**
- Füge ein: `https://github.com/JasminK11/Cannabis-Profis.de`
- Klicke **"Import"**

### 3. Deploy
- Framework: **Vite** (wird automatisch erkannt)
- Klicke **"Deploy"**

✅ **Fertig!** Deine Website ist live!

---
